# 修改图片默认宽度
为了方便部分用户，写了一个修改图片默认宽度的插件。

可在plugins\siyuan-css-img-width\index.css 中修改宽度，默认值为33%

### CSS

```css
/* 修改图片默认宽度 CSS片段 */
[data-type="img"] {
    display: inline-block !important;
    width: 33%;
  }
```
